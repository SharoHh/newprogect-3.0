# MarketHelper Dashboard Responsive

Версия с адаптивностью под смартфоны, планшеты и разные разрешения.

## Что исправлено
- мобильная верхняя панель
- sidebar на смартфонах превращается в drawer
- сетки перестраиваются в 1 колонку
- карточки и метрики адаптируются
- исправлен горизонтальный скролл
- добавлены mobile breakpoints

## Запуск

npm install
npm run dev
